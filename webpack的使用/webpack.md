全局安装webpack
cnpm install webpack@3 -g

webpack的服务：webpack-dev-server 
cnpm install webpack-dev-server@2 -g


使用webpack编译文件：
方式1：
    命令行执行：
    >webpack 资源文件路径 输出文件路径

方式2：
    配置webpack编译的配置webpack：webpack.config.js
    命令行执行：
    >webpack


webpack的功能：
1.编译模块化代码：支持AMD，CMD，common.js,es6 module

2.编译es6语法

3.编译css语法

4.编译sass
